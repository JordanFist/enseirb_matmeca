On retrouve dans ce répertoire plusieurs fichiers .java dont Autobus.java et PassagerStandard.java avec leurs interfaces Vehicule.java et Passager.java ainsi que leurs tests respectifs TestAutobus.java et TestPassagerStandard.java aidés par les fichiers Fauxpassager.java et FauxVehicule.java.

Il y a également les fichiers tests du fichier Jauge.class et Passager.class revu. 

On trouve également un Makefile, avec les commandes build, run et clean. La commande make réalise build et run. Elle lance les tests de TestJauge et TestPosition mais aussi ceux de TestAutobus et TestPassagerStandard.
